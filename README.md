# Build the project
Dependencies required :
- SDL2
- SDL2_ttf
- SDL2_image
- cmake

Build :
```bash
git clone https://github.com/Fgdou/ProjProg2022
cd ProjProg2022
mkdir build
cd build
cmake ..
make
```
Run :
```bash
./ProjProg2022
```
